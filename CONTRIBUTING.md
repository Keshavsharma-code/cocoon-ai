# 🤝 Contributing to Cocoon

We welcome all contributors! The swarm grows stronger with every new larva type and optimization.

## How to Help
1. **New Larvae**: Build specialized agents in `src/larvae/`.
2. **Hive Optimization**: Improve the Rust runtime in `chrysalis/`.
3. **Docs**: Help us clarify the metamorphosis process.

## Code Standards
- Python: PEP8, Pydantic for all schemas.
- Rust: Idiomatic, `cargo fmt` approved.
- Documentation: Always update `docs/` when changing core logic.
