Cocoon is a high-performance autonomous agent swarm designed for production-grade software engineering. It uses a **Brain (Python)** and **Muscle (Rust)** architecture to coordinate specialized agent "larvae" that safely build, test, and deploy complex systems.

## 🚀 The Hero Section
**Build a SaaS platform in 60 seconds.**
```bash
cocoon spin --name "my_startup" --max-budget $5.00
cocoon hatch --task "Build a SaaS landing page with Stripe"
cocoon run
```

## 🏗️ Architecture
- **Mother (Python)**: High-level intent parsing & task decomposition.
- **Hive (Rust)**: High-performance MCP orchestration & session management.
- **Larvae**: Specialized workers (Scraper, Coder, Designer, Tester).

## 🛡️ Guardrails (Hardened Swarm)
- **Token Budgets**: Stop the swarm before the bills arrive.
- **Human-in-the-Loop (HitL)**: Approve destructive actions manually.
- **Sandboxed Execution**: Safe trial of terminal commands.
